## Workflow

### 1. Read the Skill

```
view_file C:\Users\YZ\.antigravity\skills\skills\skills\theme-factory\SKILL.md
```

### 2. Show Theme Showcase

Display `theme-showcase.pdf` from the skill directory so the user can see all themes visually.

### 3. Ask for Selection

Wait for the user to choose a theme by number or name.

### 4. Apply the Theme

Read the chosen theme file from `themes/` directory, then apply:
- Colors to backgrounds, text, and accents
- Font pairings for headers and body
- Consistent styling throughout the artifact

### 5. Custom Theme

If no existing theme fits, generate a new one based on user description:
- Choose appropriate colors and fonts
- Name the theme descriptively
- Show for review before applying
